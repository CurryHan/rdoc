在开始之前，推荐先学习 Markdown语法，并正确安装和配置了 [Node.js](https://nodejs.org) v8.0 或以上。。


用于生成React组件库文档或简单的博客网站

```bash
npm install rdoc -g   # 安装工具
rdoc init my-project  # 初始化项目

cd my-project && npm start # 进入工程，启动服务
```
