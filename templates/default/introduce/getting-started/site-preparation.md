网站准备

## 目录结构

文档目录结构参考，默认 `README.md` 为首页，通常情况下每个组件根据目录放上 `README.md`，这个就是档期组件文档。

```
├── about
│   └── README.md
├── home
│   └── README.md
├── introduce
│   ├── README.md
│   ├── api
│   │   ├── README.md
│   │   ├── commands.md
│   │   ├── markdown-config.md
│   │   └── theme-api.md
│   ├── getting-started
│   │   ├── README.md
│   │   ├── install.md
│   │   ├── site-creation.md
│   │   └── site-preparation.md
│   └── guides
│       ├── README.md
│       ├── add-blog.md
│       ├── add-index.md
│       ├── custom-menu.md
│       ├── html.md
│       ├── insert-img.md
│       └── menu-sort.md
└── package.json
```
